# ⚛️ React Practice Lab

> A personal learning space for building and refining React fundamentals through hands-on practice.

A structured collection of React and front-end exercises created while learning modern React development.

This repository contains small projects, experiments, and refactored versions of course exercises from my learning journey.

---

## 🎯 Purpose

- Practice core React concepts in isolation
- Refactor class-based components into functional components
- Experiment with state management, effects, and component design
- Track learning progress over time

---

## 🗂️ Structure

Each folder represents a small project or exercise, numbered in chronological order (e.g., `01-project-name`, `02-another-project`).

Each project may include:

- Original version
- Refactored version
- Improved styling or responsiveness
- Notes or small experiments

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS (responsive, mobile-first)
- HTML5 semantic structure

---

## 📒 Notes

This is a learning-focused repository, not a production portfolio.

Some projects may be:

- Incomplete by design
- Experimental
- Refactored multiple times

---

## 📚 Learning Sources

Projects in this repository are inspired by and built alongside:

- [The Ultimate React Course](https://www.udemy.com/course/the-ultimate-react-course) by Jonas Schmedtmann
- [freeCodeCamp](https://www.freecodecamp.org) labs and challenges

Some projects are adapted, extended, or independently rebuilt based on these learning resources.

## 🧠 Goal

Build strong fundamentals in React by:

- Writing, breaking, and rebuilding components
- Understanding patterns deeply instead of memorizing them

---

## 🚀 Running a Project

Each project is self-contained.

```bash
cd 01-date-counter
npm install
npm run dev
```
